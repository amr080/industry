Burnable ERC20 TokenERC20 token with the following features: - Premint your total supply. - No minting function. This allows users to comfortably know the future supply of the token. - Methods that allow users to burn their tokens. This directly decreases total supply. Used to burn tokens from the supply.
VestingWalletThis contract is a vesting wallet that allows for the gradual release of native tokens (ether) and ERC20 tokens to a beneficiary over a specified duration. It calculates the amount of tokens that have already vested based on a linear vesting curve and allows the owner to release them.
Simple ERC721 NFT SaleERC721 NFT with the following features: - Built-in sale with an adjustable price. - Reserve function for the owner to mint free NFTs. - Fixed maximum supply.
GasliteTokenThe `GasliteToken` is an ERC20 token with built-in trading fees and liquidity management. It allows the owner to set buy/sell fees, manage liquidity on Uniswap, and control trading status. The contract includes functions for transferring tokens, approving allowances, and excluding addresses from fees. It also supports airdrops and has mechanisms for withdrawing tokens and ETH from the contract.
Buyback Token with FeesA premium token with most of the modern smart contract features that developers look for. With the Fully Baked token base, developers have the option of customizing to include additional tax features such as Salary Tax and Buyback Tax. Great if you plan on offering staking as a utility or you're creating a Play-to-Earn (P2E) project. This contract is commonly found on launchpads like pinksale.finance, apesale.finance, mofoLabs etc.
Basic DAOA very simple implementation of a DAO. Voting period is locked in at 1 week, and the voting delay is 1 block. There is no delay on approved proposals.
Soulbound NFTSoulbound Token: ERC721 Soulbound NFT with the following features: - Deployer can mint to recipients. - No transfer capability.
Circle USDC 2.1This contract is an upgraded version of a FiatToken contract. It introduces version 2.1, which includes an initialization function to transfer locked funds to a specified address and marks the contract as blacklisted. It also provides a function to retrieve the version string.
PudgyPenguinsThe PudgyPenguins contract is an ERC721 token contract that allows users to mint penguin NFTs. It has a maximum supply of 8888 penguins, with a maximum of 20 penguins per mint. The price per mint is 0.03 ETH. The contract supports pausing and resuming the sale, and allows the owner to withdraw funds.
